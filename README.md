> 🚀 A Deep Learning-powered Computer Vision project that detects **human faces**, predicts **gender**, and estimates **age groups** in real time using **OpenCV DNN** and pre-trained models.

---

## ✨ Features

* 👤 Real-time Face Detection
* 🚹🚺 Gender Prediction (Male/Female)
* 🎂 Age Group Estimation
* 📷 Webcam Support
* 🖼️ Image File Detection
* ⚡ Fast inference using OpenCV DNN
* 🧠 Powered by Pre-trained Deep Learning Models
* 💻 Beginner-friendly Python implementation

## 🚀 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sudeepsnandyal/Gender-and-Age-Detection.git
cd Gender-and-Age-Detection
```

### 2️⃣ Install Dependencies

```bash
pip install opencv-python
```

---

## ▶️ Run the Project

### 📷 Using Webcam

```bash
python gad.py
```

### 🖼️ Using an Image

```bash
python gad.py --image girl1.jpg
```

---

## 📸 Sample Output

```text
👤 Gender : Female
🎂 Age    : 25-32
```

The detected face will be highlighted with a bounding box and labeled with the predicted gender and age group.

* 📈 Predictions are estimates and may vary.
